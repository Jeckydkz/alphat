Line Alphat JS

# require node >= v8.x.x
check your nodejs version
`node -v`
[upgrade nodejs](https://google.com/)


# Running ( server ubuntu) 
- sudo apt-get install nodejs
- sudo apt-get install npm
- git clone https://github.com/Jeckydkz/alphat
- cd alphat
- npm i
- npm start

# Running (  termux )
- pkg install nodejs
- pkg install npm
- git clone https://github.com/Jeckydkz/alphat
- cd alphat
- npm install
- npm start
